# 할리스 커피 메뉴 크롤링 미니 프로젝트 
내가 할리스 커피에서 즐겨 마시는 제주레몬애플티의 칼로리가 얼마일까 궁금해져서 사이트에 방문했다가, 모든 음료 메뉴에 대한 정보를 크롤링해 엑셀 파일로 정리해 보았다.

## 사이트 구조
먼저 내가 원하는 정보를 얻기 위해서는 다음과 같은 경로를 따라가야 한다.

이제 내가 해야 할 일은 **크롤링 코드를 작성해 줄 도구(예: ChatGPT나 GitHub Copilot)**에게 사이트 구조와 URL 정보를 알려주는 것이다.  
즉, “어떤 경로로 들어가야 원하는 데이터를 가져올 수 있는지”, “각 메뉴 상세 페이지는 어떤 규칙으로 열리는지”를 설명해 줘야 모델이 올바른 크롤링 코드를 제안할 수 있다.

## HTML 구조 확인하기
<img width="1916" height="956" alt="image" src="https://github.com/user-attachments/assets/a167f71c-d786-470e-924c-b8f49af13d5b" />

필요한 정보에 마우스를 올리고 **우클릭 → 검사**를 눌러보자.  

<img width="1912" height="977" alt="image" src="https://github.com/user-attachments/assets/035caf72-7a9e-406a-b6e4-bed94123497e" />

그러면 화면 우측에 **개발자 도구(DevTools)**가 열리는데, 마우스 커서를 HTML 코드 줄 위로 옮길 때마다 해당 요소가 왼쪽 웹페이지 화면에 하이라이트된다.  
HTML을 전혀 몰라도 “아, 이 코드가 이 부분을 의미하는구나” 하고 감을 잡을 수 있다.  
(본인도 HTML에 대해서는 아는 게 전무하다…)

<img width="1896" height="757" alt="image" src="https://github.com/user-attachments/assets/bd4d15bd-a763-4ad4-ae3f-672b9cd5b80a" />
그 다음 해당 코드 줄을 우클릭 한후 사진과 같이 'copy element'선택, 복사한 내용을 ChatGPT 같은 도구에게 붙여넣기 한다.
사실 사이트에 대한 모든 정보를 줄 수 있으면 좋겠지만, 토큰 제한으로 모든걸 복사해올 수는 없으니, 작은 요소 몇개를 알려준다.
이 부분은 말로하기는 좀 어려운데, 몇 번 하다보면 감이 잡힌다.
<img width="1915" height="933" alt="image" src="https://github.com/user-attachments/assets/74a7354d-36bc-4a5c-b919-5a649e33ee35" />
우리가 최종적으로 얻고자 하는 정보가 담긴 표에 대한 내용도 복붙해주자.

## URL 구조 확인하기

이제 HTML 요소뿐 아니라 **URL 패턴**도 알아야 한다.  
<img width="1912" height="969" alt="image" src="https://github.com/user-attachments/assets/e3478bbc-cdbf-4f0c-ae61-a12e567c807e" />
개발자도구에서 network 선택하면 아래 많은 요소들이 보인다. 만약 보이지 않는다면 개발자 도구를 켠 채로 사이트를 새로고침(F5) 해보자.

---
<img width="1917" height="970" alt="image" src="https://github.com/user-attachments/assets/03646cf0-d578-409e-884c-4b7b6cde20dc" />
이 중에 우리에게 필요한 요소를 찾아야한다. 아무 요소나 누른 후, preview를 눌러보자. 대부분 알 수 없는 코드나, 페이지를 이루고 있는 수많은 이미지들이 보이는데
화면과 같이 사이트의 구조와 동일한 구조를 가진 요소를 찾으면 된다.  

---
<img width="1904" height="958" alt="image" src="https://github.com/user-attachments/assets/5a8a9413-2b38-47ec-bf1b-cf5bed84afd4" />
찾았다면 다시 header를 누른 후, URL 관련 정보를 드래그, 복사하여 역시 ChatGPT에게 넘겨준다.  
이런 식으로 URL 규칙을 파악해두면, 코드에서 카테고리를 딕셔너리 형태로 정리해서 반복문으로 돌릴 수 있다.  
즉, 사람이 하나씩 눌러 들어가지 않아도, 프로그램이 자동으로 모든 카테고리를 순회하며 크롤링할 수 있게 되는 것이다.
